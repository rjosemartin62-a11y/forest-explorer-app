Forest Explorer

Aplicación Turística Offline-First

Explora destinos turísticos sin límites

Forest Explorer es una aplicación móvil desarrollada en Flutter enfocada en la exploración turística, planificación de rutas, navegación interactiva y simulación de recorridos en tiempo real.

Descargar APK

Modo Offline-First

La aplicación funciona utilizando almacenamiento local, permitiendo acceso incluso sin conexión constante a internet.

Mapas Interactivos

Explora destinos turísticos utilizando Google Maps con un diseño visual personalizado estilo Forest Explorer.

Rutas Inteligentes

Genera trayectos reales siguiendo carreteras y caminos mediante sistemas de routing integrados.

Simulación de Viaje

Visualiza recorridos mediante simulación animada antes de realizar el trayecto real.

Tecnologías Utilizadas

Flutter

Hive Database

Google Maps SDK

Geolocator
